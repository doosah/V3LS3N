# V3LS3N Warehouse Management System

Современная система учёта и аналитики складских операций.

## 🚀 Особенности

- 📊 Операционные показатели в реальном времени
- 👥 Управление персоналом
- 📱 Адаптивный дизайн для всех устройств
- 🔄 Синхронизация с Supabase
- 📈 Сводные таблицы и аналитика

## 🛠 Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Supabase (PostgreSQL)
- **Styling**: CSS Grid, Flexbox, CSS Variables
- **Deployment**: GitHub Pages

## 📦 Установка и запуск

1. Клонируйте репозиторий:
\`\`\`bash
git clone https://github.com/your-username/v3ls3n-dashboard.git
\`\`\`

2. Установите зависимости:
\`\`\`bash
npm install
\`\`\`

3. Запустите локальный сервер:
\`\`\`bash
npm run dev
\`\`\`

## 🌐 Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы проекта
3. В настройках репозитория включите GitHub Pages
4. Выберите ветку \`main\` для публикации
