# Интеграция с Telegram ботом

## Настройка

1. **Токен бота:**
   - ✅ Токен уже добавлен в `src/js/telegram-config.js`
   - Токен: `8241855422:AAG7yW4NT5yoOagAo7My6bXDCdOo-pAhUa8`

2. **Получите ID чата:**
   - Отправьте боту любое сообщение
   - В консоли браузера выполните: `getChatId()` (функция уже создана)
   - Или получите через API: `https://api.telegram.org/bot8241855422:AAG7yW4NT5yoOagAo7My6bXDCdOo-pAhUa8/getUpdates`
   - Добавьте ID в `src/js/telegram-config.js`:
   ```javascript
   CHAT_ID: 'YOUR_CHAT_ID_HERE' // Замените на полученный ID
   ```

3. **Настройте список ответственных:**
   - В `src/js/telegram-config.js` заполните `RESPONSIBLE_PERSONS`:
   - Формат: название склада (точно как в WAREHOUSES) → Telegram логин для тегания
   ```javascript
   export const RESPONSIBLE_PERSONS = {
       'АРХАНГЕЛЬСК_ХАБ_НАХИМОВА': '@username_arkhangelsk',
       'МУРМАНСК_ХАБ_ОБЪЕЗДНАЯ': '@username_murmansk',
       // ... и т.д. для всех складов
   };
   ```
   - **ВАЖНО:** Названия складов должны точно совпадать с названиями в `config.js` (WAREHOUSES)

## Расписание отправки

- **Дневная смена:**
  - 07:45 - Напоминание о незаполненных отчетах (тегание ответственных)
  - 08:00 - Финальный отчет (если все заполнено до 07:45, то в 07:45 не отправляется)

- **Ночная смена:**
  - 21:45 - Напоминание о незаполненных отчетах (тегание ответственных)
  - 22:00 - Финальный отчет (если все заполнено до 21:45, то в 21:45 не отправляется)

## Примечания

- Время указано по московскому времени (Europe/Moscow)
- Для работы планировщика нужен серверный компонент (Node.js backend)
- В браузере планировщик работает только когда страница открыта
