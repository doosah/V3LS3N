# Инструкция по обновлению плашки с датой последнего обновления

## Как обновить дату/время на плашке:

При каждом функциональном обновлении сервиса нужно обновить дату в **двух местах**:

### 1. index.html (строка ~293)
```html
<span id="updateBadgeText">Последнее обновление функционала ДД.ММ.ГГГГ ЧЧ:ММ:СС</span>
```

### 2. src/js/app.js (строка ~761)
```javascript
const lastUpdateDate = 'ДД.ММ.ГГГГ ЧЧ:ММ:СС';
```

## Формат даты:
- **ДД.ММ.ГГГГ ЧЧ:ММ:СС**
- Например: `06.11.2025 03:22:21`
- Время указывается по московскому времени (MSK)

## Важно:
- Обновляйте дату **только при функциональных изменениях**
- Исправления багов и мелкие правки можно не указывать
- Дата должна отражать момент последнего значимого обновления функционала

## Автоматизация (опционально):
Можно создать скрипт для автоматического обновления даты перед деплоем:

```powershell
# update-deploy-date.ps1
$date = Get-Date -Format "dd.MM.yyyy HH:mm:ss"
# Обновить в файлах index.html и src/js/app.js
```
